# IS 312 – Activity 3: Basic Web App Skeleton

## Framework Used
Flask

## Description
This project is a simple Flask web application created for IS 312 – Activity 3.
It displays a "Hello World! + Gay Marie R. Pil" message when accessed through a browser.

## File Structure
- app.py – Main Flask application file
- README.md – Project documentation

## How to Run the Application Locally

1. Make sure Python is installed.
2. Open a terminal in the project folder.
3. Install Flask (pip install flask):
4. Run the application (python app.py):
5. Open a browser and go to http://127.0.0.1:5000:
